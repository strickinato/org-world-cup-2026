Use the following function to change the timezone.  E.g., set =n= to +6 hours for
Central European Summer Time (UTC+2).  Copy code block to org-world-cup-2026.org and use =C-c= to eval it.

#+HEADERS: :var n=-1
#+begin_src emacs-lisp :results silent
(save-excursion
 (goto-char (point-min))
  (while (re-search-forward "[[<]" nil t)
    (when (org-at-timestamp-p t)
      (org-timestamp-change n 'hour))))
#+end_src
